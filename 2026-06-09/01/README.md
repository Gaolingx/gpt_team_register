# gpt注册机使用说明

1.注册机让DeepSeek或者破限的gpt-5.4帮你配置安装依赖并运行
2.只需要配置一个动态代理或者自己改一下只需要一个代理 不用动态代理

3.软件取的rt可能二十来个请求就401掉线了  但是可以用功能：python ChatGPT_team.py --check-tokens 进行续签覆盖源文件

4.默认导出cpa格式到运行目录codex_tokens，去这个网站转sub2api格式：https://gtxx3600.github.io/CPA2sub2API
5.示例运行命令：python ChatGPT_team.py --total 10000 --workers 20
